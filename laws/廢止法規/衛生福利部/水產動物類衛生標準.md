# 水產動物類衛生標準

**法規類別**：廢止法規＞衛生福利部

**異動日期**：2019/08/15  

**法規網址**：https://law.moj.gov.tw/LawClass/LawAll.aspx?pcode=L0040088

**已廢止**：是



## 法條
##### 第 1 條
本標準依食品衛生管理法第十七條規定訂定之。

##### 第 2 條
1. 水產動物可食部分中重金屬含量應符合以下限量（以濕重計）：  
┌───────────┬──────┬─────┬─────┐  
│                項目  │甲基汞      │鎘        │鉛        │  
│類別                  │            │          │          │  
├───────────┼──────┼─────┼─────┤  
│鯨、鯊、旗、鮪魚、油魚│2 ppm 以下  │0.3ppm以下│0.3ppm以下│
1. ├───────────┼──────┼─────┼─────┤  
│鱈魚、鰹魚、鯛魚、鯰魚│1 ppm 以下  │          │          │  
│、　　魚、嘉　魚、比目│            │          │          │  
│魚、烏魚、魟魚、帶魚、│            │          │          │  
│　、　、烏鯧、鰔、鱘魚│            │          │          │  
│、金錢魚、鰻魚、梭子魚│            │          │          │
1. ├───────────┼──────┼─────┼─────┤  
│其他魚類              │0.5 ppm 以下│          │          │  
├───────────┼──────┼─────┼─────┤  
│貝類                  │0.5 ppm 以下│2 ppm 以下│2 ppm 以下│  
├───────────┼──────┼─────┼─────┤  
│頭足類（去除內臟）    │0.5 ppm 以下│2 ppm 以下│1 ppm 以下│  
├───────────┼──────┼─────┼─────┤  
│甲殼類                │0.5 ppm 以下│0.5ppm以下│0.5ppm以下│  
└───────────┴──────┴─────┴─────┘

##### 第 3 條
本標準自發布日施行。
